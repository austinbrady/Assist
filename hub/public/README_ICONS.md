# Icons

To create the required icons, you can use any image editor or online tool:

1. **icon-192.png**: 192x192 pixels, PNG format
2. **icon-512.png**: 512x512 pixels, PNG format  
3. **favicon.ico**: 32x32 pixels, ICO format

Or use this simple approach:
- Use any AI logo or create a simple gradient background with "AI" text
- Recommended background color: #667eea (purple)
- Text color: white

The icons are optional - the app will work without them, but you may see 404 errors in the console.
